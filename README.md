### # ABOUT THIS APP
Label Map adalah aplikasi berbasis web yang dirancang untuk membantu para pengguna dalam mencari lokasi yang mempunyai sertifikasi seperti halal, ISO, dan sebagainya.

### `Features Available`

* Search
* Filter
* Sertifikasi

### `Prerequisites`

- Before you begin, ensure you have met the following requirements:
  - NPM as package manager
  - Reactjs
  
### `How To Install` 
```bash
1. git clone https://github.com/panjiteknologi/project-peta.git
2. cd folder project-peta;
3. npm install;
4. npm start;

```
